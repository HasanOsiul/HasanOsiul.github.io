01 — Project Overview

Mathematical Modeling &
Special Constraint Types

This project demonstrates how managers and analysts use mathematical models with special constraint types to make optimal, real-world decisions across industries.

⚙️

Big-M Formulation

A technique that uses a large constant M with binary variables to activate or deactivate constraints, enabling conditional logic within an optimization model.

⇄

Either-Or Constraints

The model must choose exactly one feasible region from two alternatives — Region F1 or Region F2 — but never both simultaneously.

✦

K-out-of-N Constraints

Select exactly K alternatives from N available options. Mathematically enforced through binary variable summation: y₁ + y₂ + ... + yₙ = K.

💰

Fixed Charge Constraints

Certain activities can only occur after paying a fixed setup cost. Examples: opening a factory, developing an aircraft, activating a production line. If development is skipped, production must remain exactly zero.

Demo 01
Big-M Constraint Simulator
Toggle binary variable y and observe constraint activation
Binary Variable y:
y = 0
Constraint 1:  x ≤ 4 + M·(1 − y)
Activated
Constraint 2:  x ≥ 7 − M·y
Deactivated
When y = 0: Constraint 1 becomes x ≤ 4 + M (effectively no upper bound — deactivated), and Constraint 2 becomes x ≥ 7 (strict lower bound — activated). The Big-M term pushes inactive constraints into the feasible region so they never bind.
💡 Big-M Insight: When y = 0 → C1 is redundant, C2 binds. When y = 1 → C1 binds at x ≤ 4, C2 is redundant. A single binary variable controls two constraints.

02

Big-M Formulation

x ≤ 4 + M(1 − y)
x ≥ 7 − M·y
y ∈ {0, 1}, M = very large number

Binary variable y acts as a switch. When y = 0, the M term deactivates Constraint 1 and activates Constraint 2. When y = 1, the opposite occurs.
